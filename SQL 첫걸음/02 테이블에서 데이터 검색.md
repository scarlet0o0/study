# 테이블에서 데이터 검색

<br/>

<br/>

## 데이터 찾기

```sql
SELECT * FROM 테이블명;
```

<br/>

## 테이블 구조 참고하기

```sql
DECS 테이블명;
```

<br/>

## 검색조건 지정하기

```sql
SELECT 열1, 열2 FROM 테이블명 WHERE 조건식
```

<br/>

### 검색조건식

문자열을 비교할 경우 ' ' 로 둘러싸서 표기하자

- = : 같은지 비교
- <> : 서로 다른지 비교
- IS NULL : 널인 경우 true
- ">"
- ">="
- <
- <=

### 조건 추가하기

우선수위는 () 로 묶기

- AND
- OR
- NOT

<br/>

### 패턴 매칭의 의한 검색

```sql
WHERE 열명 LIKE '패턴' 
```

- % : 임의의 문자열
- _ : 임의의 문자하나
