## 스트림이란 무엇인가?

자바 8에 새롭게 추가된 기능이며 선언형으로 컬렉션데이터를 처리할 수 있게 도와주는 API이다. 

- 선언형 : 더 간결하고 가독성 상승
- 조립할 수 있음 : 유연성이 좋아진다.
- 병령화 : 성능이 좋아진다.

<br/>

<br/>

## 스트림 시작하기

스트림 -> **데이터 처리 연산**을 지원하도록 **소스**에서 추출된 **연속된 요소** 

주요특징

1. 파이프라이닝

   - 게으름

   - 쇼트셔킷

2. 내부 반복

## 스트림과 컬랙션

| 스트림                    | 컬렉션                                  |
| ------------------------- | --------------------------------------- |
| 데이터를 처리 하는데 중점 | 데이터를 보관하는데 중점                |
| 요청할때만 요소를 계산    | 모든 요소는 컬렉션에 추가되기 전에 계산 |
| 내부 반복                 | 외부 반복                               |
| 한번 만 탐색 가능         | 여러번 탐색 가능                        |
